BFS和DFS

深度优先搜索和广度优先搜索属于一种图搜索算法，它避免了图的搜索陷入了死循环。

深度优先遍历

深度优先遍历主要采用了递归算法，它的主要思想是任意确定一个顶点A，假设顶点A的邻接顶点有B、C、D，从A的邻接顶点中选取一个顶点作为遍历点，假设选择了B，那么再从B的邻接顶点中选择一个顶点遍历，依次递归。同时需要构造一个数据结构维护点是否被遍历过的状态。假设有一个点的所有邻接顶点全部遍历完成，那么退回到这个顶点的上一个顶点继续选择顶点遍历。

其实深度优先搜索采用递归的算法就是利用了栈的特性。如果不使用递归，采用的栈的话，搜索过程如下：

  1. 将根节点加入栈中
  2. 弹出节点，将该节点的所有邻接节点加入栈中
  3. 不断重复步骤2，即弹出节点的下一级节点加入栈中
  4. 如果栈为空，说明搜索已完成

广度优先遍历

图的广度优先遍历在视觉上其实是一种发散型的遍历，也就是从一个顶点散开到该顶点的所有邻接点。这就需要使用一个队列维护散开顶点。

具体的步骤如下：

  1. 根节点放入队列
  2. 从队列中弹出节点，将该节点的所有邻接节点（下一级节点）全部放入队列
  3. 不断重复步骤2，即将弹出的节点的邻接节点全部加入队列中
  4. 如果队列为空，说明已经搜索完成
